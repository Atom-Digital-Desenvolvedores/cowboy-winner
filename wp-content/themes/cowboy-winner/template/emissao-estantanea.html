Software

Software para gerenciamento de impressão com recursos poderosos e fácil de operar. É totalmente parametrizável e configurável para edição e impressão de etiquetas, cartões, crachás, carteirinhas ou quaisquer documentos em geral, suporta dados fixos, banco de dados (ACCESS, XLS, VFP, ORACLE, MYSQL, ETC.), arquivo de dados texto (delimitado ou colunado), Foto, Imagem, Smart Card(Mifare), Código de barras, Tarja magnética, Contadores automáticos, Frente e verso casados, Filtragem e seleção dos dados e vários outros recursos.
